# 知产管家 · IP Keeper v2.0

知识产权全生命周期管理平台 —— 本地运行，数据安全

## 📁 文件说明

```
知产管家/
├── 知产管家.app           ← 推荐双击这个启动（无窗口）
├── 启动知产管家.command   ← 兼容入口
├── server.py              ← 后端服务器（勿删）
├── ipkeeper.db            ← 数据库文件（自动生成）
├── uploads/               ← 上传的证书文件（自动生成）
└── static/
    └── index.html         ← 前端界面（勿删）
```

## 🚀 使用方法

### 首次使用

1. 将整个 `知产管家` 文件夹放到你想要的位置（如「文档」文件夹）
2. 双击 `知产管家.app`
3. 如提示"无法打开"，右键 → 打开 → 点击"打开"（仅首次需要）
4. 等待浏览器自动打开，即可使用

### 日常使用

每次使用时，双击 `知产管家.app` 即可。它会在后台常驻，浏览器里直接访问 `http://localhost:5678`。

如果你更习惯旧入口，也可以继续双击 `启动知产管家.command`。

### 停止服务

在「系统设置」里点「停止服务」即可，或者结束后台进程。

## 💾 数据备份

- 数据库文件：`ipkeeper.db`（复制此文件即可备份所有数据）
- 证书文件：`uploads/` 文件夹（请一并备份）
- 也可在应用「系统设置」页面导出 JSON 备份

## 🔧 常见问题

**Q: 双击后提示"无法打开"？**
右键点击文件 → 选择「打开」→ 弹窗中点击「打开」

**Q: 浏览器显示"无法连接"？**
等待10秒后刷新页面；或检查终端窗口是否有错误信息

**Q: 提示 Flask 安装失败？**
打开「终端」应用，粘贴运行：`pip3 install flask`

**Q: 更换电脑如何迁移？**
将整个 `知产管家` 文件夹（含 ipkeeper.db 和 uploads/）复制到新电脑即可

## 📌 注意事项

- 使用过程中请保持终端窗口开启
- 数据存储在本地，不上传任何云端
- 建议定期备份 `ipkeeper.db` 文件
- 启动后不需要一直开着终端窗口，推荐直接用浏览器访问
